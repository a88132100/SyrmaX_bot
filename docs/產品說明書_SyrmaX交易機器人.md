# SyrmaX 自動加密貨幣交易機器人 - 產品說明書

## 📋 產品概述

**SyrmaX** 是一款專業級的加密貨幣自動交易機器人系統，採用先進的技術指標策略和智能風控機制，為用戶提供安全、穩定、高效的自動化交易解決方案。

### 🎯 產品定位
- **目標用戶**: 加密貨幣投資者、量化交易愛好者、專業交易員、跨資產投資者
- **核心價值**: 讓一般用戶能夠使用「策略包」秒級執行多幣種、跨資產類別自動交易
- **技術特色**: 支援多交易所、多資產類別、完善風控、即時監控、智能稽核

### 策略執行流程

#### 1. 數據預處理
```python
def _precalc(df, cfg):
    # 計算所有策略共用的技術指標
    df["rsi"] = talib.RSI(df["close"], timeperiod=cfg["rsi_period"])
    df["atr"] = talib.ATR(df["high"], df["low"], df["close"], timeperiod=14)
    df["ema_fast"] = talib.EMA(df["close"], timeperiod=cfg["ema_fast"])
    df["ema_slow"] = talib.EMA(df["close"], timeperiod=cfg["ema_slow"])
    # ... 更多指標
```

#### 2. 策略執行
```python
def run_strategy_bundle(df, style):
    # 1. 獲取策略組合
    bundles = strategy_bundles[style]
    
    # 2. 逐一執行策略組合
    for bundle in bundles:
        signals = []
        for strategy in bundle["strategies"]:
            result = strategy(df)
            signals.append(result)
        
        # 3. 投票決策
        decision = _vote(signals, style)
        if decision != 0:
            return decision
    
    return 0  # 無信號
```

#### 3. 投票機制
```python
def _vote(sides, style):
    if style == "aggressive":
        # 激進：任一策略同意且無反對即執行
        if 1 in sides and -1 not in sides:
            return 1
        if -1 in sides and 1 not in sides:
            return -1
    
    elif style == "balanced":
        # 平衡：60%以上策略同意
        pos, neg = sides.count(1), sides.count(-1)
        if max(pos, neg) / len(sides) >= 0.6:
            return 1 if pos > neg else -1
    
    elif style == "conservative":
        # 保守：全部策略同向才執行
        if all(s >= 0 for s in sides) and any(s == 1 for s in sides):
            return 1
        if all(s <= 0 for s in sides) and any(s == -1 for s in sides):
            return -1
    
    return 0
```

### 策略性能特點

#### 激進模式特點
- **高頻交易**: 平均每小時1-3次交易機會
- **快速反應**: 3-8週期EMA捕捉短期趨勢
- **高風險高收益**: 適合波動性大的市場
- **適用場景**: 趨勢明確、成交量活躍的市場

#### 平衡模式特點
- **中等頻率**: 平均每2-4小時1次交易機會
- **穩健獲利**: RSI均值回歸+ATR突破組合
- **風險可控**: 適合大部分市場環境
- **適用場景**: 一般市場環境，尋求穩定收益

#### 保守模式特點
- **低頻交易**: 平均每天1-2次交易機會
- **高勝率**: 多重確認才進場
- **低風險**: 適合資金保護
- **適用場景**: 趨勢不明確或高風險時期

### 策略優化機制

#### 參數自適應
- 根據市場波動率動態調整ATR倍數
- 根據成交量變化調整策略權重
- 根據歷史表現調整策略組合

#### 市場環境識別
- 趨勢市場：使用順勢策略
- 震盪市場：使用均值回歸策略
